3.) Open up puttygen.exe


Click on Generate (blue box), move your mouse around. DO NOT CLOSE THE WINDOWN WHEN FINNISHED GENERATING KEY. Copy the Public key from the textarea and send it to me on mail (red box) marius_flagstad@hotmail.com. When I add this key to the repo you will be able to access it.


4.) Save the keys on your computer

Not that I have your public key, you need to click on the save private key (yellow box) and add it in your home area under a folder called ".ssh". The easiest way to find your homearea is to type windowskey + R, type inn cmd, and see where you start. You can here type mkdir .ssh and save your private key to this folder (I call it id_rsa.ppk), password protection for your key is not needed (will be prompted when you try to save), but you can add it if you want to (I don't)

It is also recommended to save your public key under the same folder so that you have it if you need to give it out to some other places. Now you can close the puttygen.exe window


5.) Get the git repo

If I now have added your key you should be able to download it. Open GIT Bash and go to a location where you want your reposetory to be on your computer (cd C:/blabla/)

Dont create a folder where you want your repo to be. The repo will allready be contained withing a folder that is named Wasteland. If you are on C:/ you will end up with C:/wasteland as your repo folder.

Type inn git clone git@github.com:Mabluz/wasteland.git

GIT TIPS (must be on the repo folder with git bash)

--------- THESE COMMANDS ARE DONE BY THE "COMMIT ALL WORK TO GIT.bat" FILE ---------

git pull - downloads the files that have been changed by others

git status - shows you a list over all your local files that you have changed and not commited to the repo yet

git add . - Adds all your files shown by status to the commit stash

git commit -am "MY MESSAGE" - Commit your files that are added

git push - pushes out all your commited files

--------- END BAT FILE COMMANDS ---------

gitk - Get up a GUI view over all commits added by other people
